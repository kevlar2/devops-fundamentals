# Writing Scripts with Variables and Input
## Objective 

The purpose of this task was to showcase my knowledge of Bash scripting. I developed an interactive and personalized Bash script that, for example, dynamically displays the current date, time, and user's name.

### Acceptance Criteria
 - The script runs without errors.
 - The script uses read to capture input.
 - The greeting message is personalized (e.g., if the user types “Afi”, the script should say: Hello Afi!).
 - The date and time appear in the output dynamically.
 - The script must be executable (chmod +x ask_user.sh) and runnable with ./ask_user.sh.

### Deliverables
 - provide:
   - ask_user.sh → the script file.
   - commands.txt → a text file listing:
   - The commands you ran to create, edit, and execute the script.
   - A brief explanation (in your own words) of what each command does.
