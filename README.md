# Devops-Fundamentals - CloudPros DevOps Cohort

This repository contains hands-on exercises and learning materials for the CloudPros DevOps cohort. It covers fundamental DevOps concepts, tools, and practices through weekly structured learning modules.

## Project Structure

The repository is organized into weekly modules, each focusing on specific DevOps topics:

```
devops-fundamentals/
├── README.md                           # Project overview and documentation
├── week1/                              # Process Management & System Monitoring
│   ├── images/                         # Screenshots and visual documentation
│   ├── process-management/             # Process management exercises and notes
│   └── readme.md                       # Week 1 specific documentation
├── week2/                              # Bash Scripting & Automation
│   ├── bashscript-with-variables-and-input/  # Interactive bash scripts
│   └── readme.md                       # Week 2 specific documentation
└── week3/                              # Networking & DNS
    ├── task1/                          # DNS exploration and analysis
    │   └── dns-interactive-explorer/   # DNS lookup tools and reports
    └── task2/                          # Network troubleshooting
        └── networking-troubleshooting/ # Network diagnostic exercises
```

Each week contains practical exercises, documentation, and real-world examples to build foundational DevOps skills.
